# SQL Básico - Exercício 1

## Objetivo
Praticar os comandos básicos e intermediários de SQL, consolidando os fundamentos do MySQL.  
O exercício inclui criação de tabelas, inserção de dados e consultas diversas, explorando filtros, ordenações e funções agregadas.

## Descrição do Trabalho
Neste exercício, criamos a tabela `empregados` com registros de exemplo.  
Os dados foram importados da base MySQL para Python usando Jupyter Notebook, permitindo visualizar os resultados das queries de forma interativa.  

Cada questão no notebook possui **explicação em Markdown** seguida da query em Python, mostrando o resultado diretamente com Pandas.

## Principais Comandos Usados
- `CREATE TABLE` → criar tabelas  
- `INSERT INTO` → inserir registros  
- `UPDATE` → atualizar registros  
- `DELETE` → remover registros  
- `SELECT` → consultar dados  
- `WHERE` → aplicar filtros em consultas  
- `AND` / `OR` → combinar condições  
- `BETWEEN` → filtrar intervalos de valores  
- `LIKE` → buscar padrões em strings  
- `IS NOT NULL` → filtrar registros com valores não nulos  
- `ORDER BY` → ordenar registros (ASC / DESC)  
- `LIMIT` → limitar número de resultados  
- `DISTINCT` → selecionar valores únicos  
- Funções agregadas:  
  - `COUNT()` → contar registros  
  - `SUM()` → somar valores  
  - `AVG()` → calcular média  

## Aprendizados
- Criar e manipular tabelas simples no MySQL  
- Inserção, atualização e exclusão de registros  
- Estrutura de queries básicas e avançadas  
- Aplicação de filtros e condições em consultas  
- Ordenação e limitação de resultados  
- Seleção de valores únicos  
- Uso de funções agregadas para análise de dados  
- Conectar Python a MySQL via Jupyter Notebook  
- Visualização de resultados de SQL usando Pandas  

## Observações
- Cada questão do exercício está comentada no notebook, explicando o objetivo e a lógica da query.  
- O notebook permite **visualizar os resultados diretamente**, tornando o portfólio mais interativo e profissional.
